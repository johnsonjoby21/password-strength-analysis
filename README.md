# Password Strength Evaluation Lab

## Objective
To understand how password strength works and evaluate different passwords using an online password strength checker.

## Tool Used
Kaspersky Password Strength Checker

## What Was Done
- Created multiple passwords with varying complexity.
- Tested each password on an online password strength checker.
- Analyzed feedback and scores.
- Studied common password attacks.

## Key Findings
- Short passwords are weak.
- Predictable passwords are vulnerable.
- Random and long passwords are strongest.
- Multi-factor authentication increases security.

## Outcome
Gained understanding of password complexity, strength evaluation, and common password attack techniques.
